# x1 Eco Chain Testnet

1. **Set Up**

```bash
git clone https://github.com/airdropbomb/x1.git && cd x1
```

2. **Install Dependencies**

```bash
pip install -r requirements.txt
```

4. **Enter Your PrivateKey**

```bash
nano pk.txt
```

6. **If you want to use proxy , Enter Your Proxy**

```bash
nano proxy.txt
```

**Run Bot**

```bash
python bot.py
```
